# Draw & Order

Think you have what it takes to be a police sketch artist? Users recieve an description generated by AI based on an AI generated image, containing a human face with distinctive features. The image is hidden by default, and can currently be revealed via button click. Plans to add accuracy ratings and potentially competetive aspect in the future.

## Tech Stack

- Next.js App Router
- React + Konva drawing surface
- Supabase Auth + Storage
  ~~- Hugging Face inference API (text-to-image)~~
- OpenAI ChatGPT vision (text-to-image generation, and image-to-text descriptions)
